[edit] Open source

[edit] Multi-platform

cdrtools, a comprehensive command line-based set of tools for creating and burning CDs and DVDs.
cdrkit, a fork of cdrtools by the Debian project.
cdrdao, open source software for authoring and ripping of CD-ROMs in Disk-At-Once mode.


[edit] Linux and Unix

dvd+rw-tools, a package for DVD and Blu-ray Disc writing on Unix and Unix-like systems.


GUI (cdrtools based)


Brasero, a GNOME CD/DVD burning application
GnomeBaker, a GNOME CD/DVD burning application
Graveman, a GTK+ CD/DVD burning application - project abandoned
K3b, the KDE CD authoring program
Nautilus, the GNOME file manager (includes basic CD/DVD-burning capabilities)
Serpentine, the GNOME audio CD burning application
X-CD-Roast


command-line interface


BashBurn, a menu-driven program
