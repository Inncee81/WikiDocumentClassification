Hydrogen is an advanced drum machine created by Alessandro Cominu, an Italian programmer who goes by the pseudonym Comix. Its main goal is to provide professional yet simple and intuitive pattern-based drum programming.
Hydrogen was originally developed for Linux, but later ported to Mac OS X. An experimental build exists for Microsoft Windows but the port has since been abandoned.[1] The graphical user interface for the application uses Qt library. All code is released under the GNU General Public License. Hydrogen is free software.

[edit] Features

Pattern-based sequencer, with unlimited number of patterns and ability to chain patterns into a song.
Up to 64 ticks per pattern with individual level per event and variable pattern length.
32 instrument tracks with volume, mute, solo, pan capabilities.
Multi-layer support for instruments (up to 16 samples for each instrument).
Ability to import/export song files.
Unique human velocity, human time, pitch and swing functions.
Multiple patterns playing at once.
