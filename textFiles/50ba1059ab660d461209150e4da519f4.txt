This list of PDF software includes links to articles on computer software used to manage PDF documents.

[edit] Multi-platform

[edit] Converters

PdfTeX – converter (from TeX)
Prince – PDF formatter using XML or HTML with CSS and SVG as input
pstoedit – PostScript to (other) vector file format converter
XeTeX – a TeX typesetting system that creates PDF documents
SWFTools – free tools, contains tool for converting pdf to swf


[edit] Development libraries

iText – a free and open source library for creating and manipulating PDF, RTF, and HTML files in Java as well as C# and other .NET languages
JasperReports – is an open source Java reporting tool that can write to screen, to a printer or into PDF, HTML, Microsoft Excel, RTF, ODT, Comma-separated values and XML files.
PDF Clown – a free software library for creating and manipulating PDF files in Java 1.5 (or later) and .NET 2.0 (or later)
Poppler – an open source multi-backend library derived from Xpdf, without a bundled viewer, but providing a cairo backend
ReportLab – an open-source, Python-based PDF creation library from ReportLab
TCPDF - an open-source PHP library for creating PDF files
Xpdf – an open source multi-backend library for viewing and manipulating PDF files. Bundled with a viewer with the same name


[edit] Creators

Adobe Acrobat – Adobe Systems commercial desktop PDF authoring suite.
Ghostscript – viewer, creator and converter.
Inkscape – open, edit and export documents, Inkscape is free software.
PageStream – commercial cross-platform desktop publishing application to open, edit and export.
Pdftk – command-line tools to create, edit and convert documents.
Scribus – cross-platform desktop publishing (DTP) application
PDFedit – a free software (GPL) for viewing or editing the internal structures of PDF documents.
PDF Studio – a commercial software for viewing or editing PDF documents.


[edit] Viewers

Adobe Reader – Adobe's PDF reader.
GSview – a graphical interface for Ghostscript and is very helpful for extracting data from locked PDFs.
Okular – universal viewer for KDE; replaces KPDF, available on Windows from KDE 4.1
Xpdf – a multiplatform viewer.


[edit] AmigaOS

[edit] Converters

Antiword a free Microsoft Word reader for various operating systems. Antiword converts the binary files from Word 2, 6, 7, 97, 2000, 2002 and 2003 to plain text and to PostScript. Available for AmigaOS 4, MorphOS and AROS X86.
dvipdfm a DVI to PDF translator with zlib support


[edit] Creators

ClibPDF PDF Writer Linker-Lib for StormC (68k+PPC)


[edit] Viewers

gv a file viewer program included with the multi-platform Ghostscript package
Xpdf a multiplatform viewer for Portable Document Format (PDF) files. Amiga version uses X11 engine Cygnix.


[edit] Unix-like

[edit] Converters

CUPS (Common Unix Printing System) – supports exporting to PDF formats
GIMP
OpenOffice.org
Pdftk (the PDF Toolkit) – can merge, split, en-/decrypt, watermark/stamp and manipulate PDF files


[edit] Creators

Xournal – an open source program which also supports annotating pdf files


[edit] Viewers

Adobe Reader – Adobe Reader also has a freeware version for Linux
Evince – the default PDF and file viewer for GNOME; replaces GPdf
gv – a file viewer program included with the multi-platform Ghostscript package
GPdf – a PDFviewer for GNOME; replaced by Evince
KPDF – a PDF reader for the K Desktop Environment; replaced by Okular in KDE 4
Xpdf – a PDF viewer for the X Window System


[edit] Mac OS X

[edit] Converters

deskUNPDF for Mac – commercial application from Docudesk for converting PDF files to the Microsoft Office or OpenOffice.org file formats


[edit] Creators

Mac OS X's printing subsystem enables any application with printing capability to generate a PDF.


[edit] Viewers

Adobe Reader – Adobe Systems Reader is also available on the Mac platform.
Preview – Mac OS X's default PDF viewer (in v10.5 it also can annotate, rearrange pages, and merge PDF files)
Safari – Apple's web browser supports in-line PDF viewing.
Skim – an open source PDF reader and note-taker for Mac OS X


[edit] Microsoft Windows

[edit] Converters

deskUNPDF Standard and Professional – commercial PDF converter applications from Docudesk
Nuance PDF Converter – a commercial, proprietary PDF viewing, editing & conversion tool from Nuance Communications
Software602 Print2PDF – commercial PDF converter from Software602 Inc.


[edit] Creators

CutePDF Professional – commercial PDF editor
deskPDF Standard and Professional – commercial PDF converter
DoPDF – freeware PDF creator
Ecopy Desktop – commercial PDF editor
Nitro PDF – commercial PDF viewer, creator, editor and converter application
PagePlus X2 – commercial desktop publishing suite with PDF viewing, editing and creation features
PaperPort Scansoft – commercial desktop publishing suite with PDF viewing, editing and creation features
PDFCreator – free, open source PDF creator


[edit] Viewers

Adobe Reader – Adobe's PDF reader, offered at no cost but a heavyweight download and install
PDF-XChange Viewer – freeware viewer with extended/markup capabilities, installer based and portable versions.
Foxit Reader – Free basic reader, extended functionality available via purchasable plugins.
Sumatra PDF – a free, open source, light-weight PDF
GSview – Open source software Ghostscript's viewer for Windows. Unlike its sibling gv, GSview can display PDF files, not only PostScript files.


[edit] Mobile

Adobe Reader – a PDF viewer from Adobe Systems for Symbian, Palm, and Pocket PC mobile devices.
Foxit Reader – supports Windows CE 4.2 and 5.0.
PDF-XChange Viewer – freeware viewer with extended markup capabilities, comes in a portable version that can be run from any suitable device, such as a flash drive or memory stick


[edit] Web-based

[edit] Converters

PrimoOnline, free online PDF creator
Software602 Print2PDF Premium Edition – interactive web conversion from IIS or XML-SOAP web service
Zamzar – online file converter


[edit] Editors

PDFescape – a free web service which views and edits PDF documents in the browser
PDFFiller – a web-based program that allows the user to read, edit or fill out PDF forms from within the browser.
PDFVue – a free web application that allows the user to view PDF's, comment and fill PDF forms from a web browser.


[edit] Viewers

vuzit – a web service which views PDF documents (and 40 others) as HTML in the browser via AJAX.
Google docs – a web service which views PDF documents as HTML in the browser
scribd – a web service which renders PDF documents as a Flash widget within a web page
