[edit] Open source

AMANDA BSD
Areca Backup GPL - supports FTP & FTPs, compression (Zip64), encryption (Triple DES & AES), incremental backups - Java based.
BackupPC GPL
Bacula GPL - computer program suite.
Cobian Backup Mozilla Public License 1.1 - Version 8 is opensource, Version 9 not open source but no cost
cpio - Unix/Linux archiver.
DAR GPL - Disk Archiver. Special designed for differential backup. Support multi-volume compressed archives (slices).
dump - The classic UNIX filesystem dumper.
duplicity GPL - Makes remote encrypted backups using rsync and gnupg.
FlyBack GPL - rsync based, UI modeled after Time Machine (Apple software).
Mondo GPL
rsync GPL - tool for maintaining mirrors.
tar - The classic UNIX file aggregator. Work with gzip in a pipe to create tar, tar.gz (tgz) archives.
TimeVault LGPL - a Linux application modeled after Apple's Time Machine
Toucan (software) open source file synchronization, backup and encryption tool for Microsoft Windows,
Venti Open Source - developed originally as part of Plan 9 from Bell Labs but now available for unix-like systems as part of Plan 9 from User Space.
Zmanda Recovery Manager GPL - perl utility for MySQL databases.
