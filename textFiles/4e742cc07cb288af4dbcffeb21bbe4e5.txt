[edit] SIP-capable firewalls

Check Point's VPN-1 firewalls include complete SIP support for multiple vendors
Intertex SIP Transparent Routers, Firewalls and ADSL modems - For broadband deployments and the SOHO market
Ingate SIP Transparent Enterprise Firewalls and SIParators - Enabling global SIP communication on the enterprise LAN
Cisco PIX/ASA firewalls include complete SIP support
Secure Computing's SnapGear firewall includes the siproxd SIP proxy
Secure Computing's Sidewinder 7 firewall includes a SIP proxy
SonicWall Supports SIP
D-Link Firewall DFL-210/260/800/860/1600/2500 Supports SIP (SIP-ALG) with firmware 2.20.01.05 and above
ZyXEL ZyWALL P1, 2Plus, 5 UTM, 35 UTM, 70 UTM, 1050, USG 100, USG 200, USG 300, USG 1000 supports SIP-ALG
Fortinet All FortiGates running v280/v300 Builds
Linux Netfilter's SIP conntrack helper fully understands SIP and can classify (for QOS) and NAT all related traffic.
PfSense FreeBSD based PfSense Firewall has QoS that properly tags voip traffic and a sip proxy package that is available for NATed endpoints.


[edit] SIP clients

[edit] Free software

[edit] For personal computers

Ekiga, formerly known as GnomeMeeting, GPL
KCall, using Qt libraries
KPhone, using Qt libraries, GPL
Linphone, with a core/UI separation, the GUI is using GTK+ libraries
Minisip, with a core/UI separation and encryption, alpha version for Nokia 770
PhoneGaim, based on Pidgin.
QuteCom, formerly known as OpenWengo, using Qt libraries, GPL
Telephone, Mac OS X softphone written in Cocoa/Objective-C
Twinkle, using Qt libraries, GPL
SFLphone, with GTK+ GUI, GPL, also supports IAX2 protocol
SIP Communicator, the Java VoIP and Instant Messaging client, runs on Windows, Linux and Mac OS X, LGPL


[edit] Proprietary freeware

[edit] For personal computers

Gizmo5 for Windows, Mac OS X, Linux and Symbian, which allows calls to the Gizmo5 network, any SIP network, GoogleTalk and Skype via their OpenSky SIP for Skype gateway. They also offer web based Gizmocall which uses just Flash and a web browser to make calls. (Note: as of March 2008, the Gizmo linux binary is two years old, and does not run on 64-bit systems.)
Mindspring, for Windows. Voice, chatting, and file-sharing capabilities.
Nimbuzz, for Windows. Includes a native VOIP protocol and SIP capabilities amongst others.
OfficeSIP Messenger for Windows. Instant messaging, voice and video capabilities. Freeware alternative to Office Communicator
SightSpeed for Windows and Mac OS X. G.711, iLBC, Speex and H.263 codecs
Windows Messenger versions 4 and 5 (not to be confused with Windows Live Messenger or MSN Messenger which do not support SIP)
Xlite from Counterpath for Windows, Mac OS X, and Linux


[edit] For mobile phones and other devices

Fring for Linux, Java ME, Symbian OS, Windows Mobile and iPhone, supporting SIP, Skype, MSN Messenger, Google Talk, ICQ, Twitter, Yahoo! Instant Messenger and AIM
Nimbuzz supporting SIP on Windows PC, iPhone and Symbian S60.

miTester for SIP: is an open source SIP testing tool designed and developed to take care of the complex pre-deployment testing of SIP applications easily.
This SIP testing tool can be used to simulate SIP call-flows & automate functional, regression tests.
