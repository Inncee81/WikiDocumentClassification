[edit] Text readers

TextViewer supports a variety of file formats, a touch screen interface, low power mode, and persistent text bookmarking.
DSReader is another highly aesthetic text reader with True Type Font support.
DSLibris is recently released text reader incorporating HTML support.
ReadMore is alpha version of reader with book orientation.
Ands-pdf is a PDF reader.


[edit] Internet

[edit] Web Browser

DSLinux includes the web browser links and retawq.
OKIWI is a web browser for the DS in alpha stages.
DSOrganize includes a browser with support for forms and limited support for pictures.
DSHobro 0.4 requires a server on a networked PC. This server will render the webpages and then send them to the DS.
Bunjalloo is an open source web browser for the DS developed by Richard Quirk.


[edit] Instant messaging

DS-AIM for AIM
Beup Live for .NET Messenger Service
IRCDS, SylphIRC.
DSLinux includes tinyirc, BitchX, irssi
DSOrganize includes an IRC client.
JabberDS Jabber client.
ATYA is a Yahoo Messenger homebrew, but it currently has certain problems.


[edit] VoIP

HelloDS is known for its lack of compatibility. Development has ceased.
DSiP requires a VoIP account.
Infantile Paralysiser's Wifivoicechatclient allows you to draw as well as chat.
SvSIP is a VoIP application based on pjsip.


[edit] Remote administration

PointyRemote
Win2DS, a very reliable remote with two modes, one power saving.
DS2Win is an enhanced version of Win2DS
DSVNC, a VNC client
DS2Key turns the DS into a wireless joystick.
DSMouse use your DS as a wireless mouse. Some testers have noted that the DSMouse only works on a fraction of the desktop.
DSLinux includes the dropbear ssh client


[edit] Other Internet-based applications

NDSMail is an email and web feed client.
DSLiveWeather is a Weather.com client.
The test program for DSWiFi can be used for wardriving.
DSBlue can also be used for wardriving. [1]
DSFTP is an FTP server.
DS2DS is an AdHoc Wifi file transfer tool that doesn't require an access point (direct transfer between DS).
Treasures of Gaia is a Google Maps browser (Available in English and French).
DSmaps a Google maps client for the NDS.
DSLinux includes inetd, samba, ftpd, telnetd and Boa Test versions of Bittorrent software are also available


[edit] Drawing applications

Colors! is a pressure sensitive drawing application focusing on simplicity and ease-of-use.
Phidias is a Photoshop-like application for ds.
DS Notes is a note application, which is similar to an offline PictoChat.
Smoove is a pixel editor for DS.
FlickBook is an animation creator.
Animanatee is another animation creator featuring 32 levels of undo and onion skinning.
Pocket Physics is an application like crayon physics where you draw levels and then let gravity act on them.
colors [2] a good pressure sensitive paint application.


[edit] Dictionaries

NewDictS is a dictionary interpreter of the StarDict .dic dictionaries, with Voice capabilities. Initially released by phillips, which released his NewDict 7 source code, now being improved and currently edited by Sesa. A translation page can be found here.
DS Dictionary: A free dictionary for the Nintendo DS
Scrabble Assistant DS While not technically a dictionary, this tool is of great help to players of the Scrabble board letter-game from Hasbro/Mattel. It features word validation, score tracking (up to 9 players) and a powerful anagram search which finds all possible words using part or all of a given set of letters. Three databases are available: english words from "Sowpods" and "TWL", french words from the "Officiel du Scrabble v5". Discussion and support on gbadev.org forum.


[edit] Language learning

Project JDS helps you learn the basics of japanese (hiragana and katakana)
DSZhongwen helps you learn the basics of mandarin chinese


[edit] Other applications

TxtWriter is a text editor optimized for typing. Some users have difficulty due to the lack of compatibility with the NDS buttons.
NitroTracker, tracker sequencer software.
DiagnoSe is a tool for testing the hardware features of the DS.
Infantile Paralysiser's Cooking Timer is a skinnable countdown timer which plays a chosen MP3 when it reaches 0:00:00.
Eigenmath DS is a graphing calculator for the Nintendo DS. The system was designed for the manipulation of symbolic algebraic expressions and matrix operations, integration, differentiation, an advanced CAS.
graphDS is a graphing calculator application for the Nintendo DS that includes expression evaluation, graphing, a relatively simple CAS, and a number of built-in functions.
3D Graph Viewer is a tool for viewing 3D graphs.
DrummerS a drumkit app, limited mainly by the DS's screen "only noticing one point of contact at any one time".

HexenDS, Hexen for the DS. Includes touchscreen support and source code. A newer branch that supports more devices also exists.

DSQuake or QuakeDS, Quake for the DS.
QuakeIIDS, Quake II on the DS, 16MB expansion RAM pack required

AmplituDS, based on Harmonix's Amplitude for the Playstation 2.
Wolfenstein 3D, a port of the popular PC DOS game Wolfenstein 3D
Spider Solitaire DS, a clone of Windows' Spider Solitaire
SensitiveDS, a clone/remake of the C64 classic Sensitive.
Ka-BoOm, a minesweeper with a twist by Stravingo.
Setsuzoku no Puzzle, an addictive puzzle game by Stravingo based on the arcade game Match It.
StillAliveDS, a puzzle game similar to Valve's Portal, by t4ils.
DiggerDS, a classic remake of the popular PC game Digger.
Visual Novel Reader DS, reads Visual Novels which have been converted, such as Tsukihime and Fate/Stay Night.
VideoGamesHero, a GuitarHero style rhythm game with over 65 Songs, 5 Game modes, Motion Card and Guitar Grip support.
LoneWolf DS, the gamebook series brought to the DS.
ChaosDS is a tactical combat video game for the Nitendo DS based on the 1985 Spectrum game Chaos - The Battle of Wizards.
HooverManeuver (2 player) If you’ve played the vacuum cleaner challenge in Wario Ware on the GBA, you know exactly what to expect. This re-implementation allows two players to compete simultaneously on 1 DS. Two Hoovers race to collect 50 items of rubbish, bouncing off each other and walls. They rotate until the press of the ‘Left’ button for one Hoover, and the ‘A’ button for the other (which causes them to travel forward in a straight line).
Population Tire, a port of a video game by Videlectrix, of Homestar Runner fame.
